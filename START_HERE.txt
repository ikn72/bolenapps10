🚀 MULAI DARI SINI - BOLEN PISANG APP TO EXE
==============================================

Selamat datang! File ini akan memandu Anda mengubah 
aplikasi web Bolen Pisang menjadi file executable (.exe).

📋 LANGKAH MUDAH (PILIH SALAH SATU):
====================================

🔥 CARA TERCEPAT (OTOMATIS):
---------------------------
1. Double-click file: "install-nodejs.bat"
2. Ikuti instruksi yang muncul
3. Setelah Node.js terinstall, jalankan "build.bat"
4. Selesai! File .exe ada di folder "dist"

⚡ CARA MANUAL:
--------------
1. Install Node.js dari: https://nodejs.org/
2. Double-click file: "build.bat"
3. Tunggu proses selesai
4. File .exe ada di folder "dist"

📁 FILE PENTING:
===============
📦 install-nodejs.bat  → Install Node.js otomatis
🔨 build.bat          → Build aplikasi jadi .exe
🔧 build.ps1          → Alternatif PowerShell
📖 INSTRUKSI_BUILD.txt → Panduan lengkap
📋 README.md          → Dokumentasi teknis

🎯 HASIL AKHIR:
==============
Setelah build berhasil, Anda akan mendapatkan:

📁 dist/
├── 📦 Bolen Pisang Interactive Setup 1.0.0.exe
└── 📁 win-unpacked/
    └── 🚀 Bolen Pisang Interactive.exe

💡 TIPS:
========
- File Setup.exe = Installer (install ke komputer)
- File Portable.exe = Langsung jalan tanpa install
- Ukuran sekitar 150-200 MB (normal untuk Electron)
- Antivirus mungkin false positive (aman)

❓ BUTUH BANTUAN?
================
Baca file "INSTRUKSI_BUILD.txt" untuk panduan detail
atau "README.md" untuk info teknis.

🎉 SELAMAT MENCOBA!
==================
